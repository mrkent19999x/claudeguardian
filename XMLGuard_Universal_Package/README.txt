========================

NỘI DUNG PACKAGE:

- config.json (Cấu hình)
- Install.bat (Cài đặt tự động)

HƯỚNG DẪN CÀI ĐẶT:
1. Right-click "Install.bat"
2. Chọn "Run as administrator"
3. Chờ cài đặt hoàn tất
